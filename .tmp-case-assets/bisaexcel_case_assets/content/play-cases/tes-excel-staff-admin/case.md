# Tes Excel Staff Admin

## Tujuan
Melatih kemampuan menyelesaikan soal tes kerja Excel yang sering muncul untuk posisi admin.

## Konteks
Kamu sedang mengikuti latihan tes Excel untuk posisi staff admin. Fokusnya adalah ketelitian dan rumus dasar.

## Rules
1. Kerjakan soal seperti simulasi tes Excel staff admin.
2. Gunakan rumus dasar dan fungsi rekap sederhana.
3. Isi jawaban akhir di web BisaExcel.
4. Tidak ada minus untuk jawaban salah.
5. Jawaban detail tersedia sebagai pembahasan premium nantinya.

## Dataset
Dataset berisi customer, kota, produk, qty, harga, status pembayaran, dan kode sales.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
