# Growth Omzet Penjualan

## Tujuan
Melatih kemampuan menghitung growth omzet, membandingkan target, dan membaca performa sales.

## Konteks
Kamu diminta membandingkan omzet 2025 dan 2026 untuk melihat pertumbuhan penjualan.

## Rules
1. Bandingkan omzet 2025 dan 2026.
2. Gunakan rumus persentase growth dengan benar.
3. Isi jawaban akhir di web BisaExcel.
4. Format persen boleh memakai titik atau koma desimal.
5. Kunci jawaban tidak tersedia di file public.

## Dataset
Dataset berisi bulan, region, produk, omzet 2025, omzet 2026, dan target 2026.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
