# Upah dan Lembur Karyawan

## Tujuan
Melatih kemampuan menghitung upah lembur dan membuat rekap data HRD sederhana.

## Konteks
Kamu berperan sebagai admin HRD yang diminta mengecek data kerja dan lembur karyawan.

## Rules
1. Baca data absensi dan lembur dengan teliti.
2. Hitung upah lembur berdasarkan Jam Lembur x Tarif Lembur/Jam.
3. Isi jawaban akhir di web BisaExcel.
4. Jawaban salah tidak mengurangi poin.
5. Kunci jawaban tidak tersedia di file public.

## Dataset
Dataset berisi tanggal, ID karyawan, nama, divisi, jam kerja, jam lembur, tarif harian, tarif lembur, dan status hari.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
