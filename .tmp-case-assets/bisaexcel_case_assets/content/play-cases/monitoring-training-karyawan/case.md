# Monitoring Training Karyawan

## Tujuan
Melatih kemampuan membuat rekap training, menghitung completion rate, dan membaca progress peserta.

## Konteks
Kamu berperan sebagai admin HRD yang diminta membuat monitoring training karyawan.

## Rules
1. Baca data training dan status peserta dengan teliti.
2. Gunakan rumus rekap untuk menghitung kehadiran, kelulusan, dan rata-rata nilai.
3. Isi jawaban akhir di web BisaExcel.
4. Tidak ada minus untuk jawaban salah.
5. File public tidak berisi kunci jawaban.

## Dataset
Dataset berisi peserta training, divisi, jenis training, tanggal, status hadir, pre-test, post-test, dan status lulus.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
