# Lookup Data Karyawan

## Tujuan
Melatih kemampuan lookup data karyawan menggunakan VLOOKUP, XLOOKUP, dan COUNTIF.

## Konteks
Kamu diminta mengambil informasi karyawan berdasarkan ID Karyawan dari master data.

## Rules
1. Gunakan ID Karyawan sebagai kunci pencarian utama.
2. Kerjakan lookup di Excel/Google Sheets.
3. Isi jawaban akhir di web BisaExcel.
4. Kunci jawaban tidak ada di file public.
5. Gunakan rumus yang paling sesuai dengan kasus.

## Dataset
Dataset berisi ID karyawan, nama, divisi, jabatan, gaji pokok, tanggal masuk, status, dan manager.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
