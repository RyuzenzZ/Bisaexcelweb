# Tes Excel Staff Admin

Folder ini berisi aset konten internal untuk studi kasus `tes-excel-staff-admin`.

Isi folder:
- `case.md`: narasi kasus, rules, dan konteks.
- `dataset.csv`: dataset dummy realistis untuk pengembangan.
- `questions.json`: daftar pertanyaan yang boleh dipakai frontend/backend.
- `answer-key.internal.json`: kunci jawaban internal. Jangan taruh di public frontend.
- `rubric.md`: rubrik challenge dan kriteria portfolio.
