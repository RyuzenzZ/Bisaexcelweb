# Rekap Penjualan Harian

## Tujuan
Melatih kemampuan membaca data transaksi dan membuat rekap penjualan sederhana.

## Konteks
Kamu berperan sebagai admin sales yang diminta membuat rekap penjualan harian dari dataset transaksi.

## Rules
1. Buka sheet Dataset dan pahami struktur data transaksi.
2. Kerjakan perhitungan di Excel atau Google Sheets.
3. Isi jawaban akhir di web BisaExcel.
4. Tidak ada pengurangan poin untuk jawaban salah.
5. Kunci jawaban tidak tersedia di file latihan.

## Dataset
Dataset berisi tanggal transaksi, invoice, sales, produk, kategori, wilayah, qty, harga satuan, dan diskon.

## Output yang Diharapkan
User bisa mengerjakan file kasus di Excel/Google Sheets, menghitung hasil akhir dengan rumus, lalu memasukkan jawaban ke web BisaExcel.
